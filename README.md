# State Tracker (SillyTavern extension)

Generic structured-state tracker for SillyTavern. Outfits, stats, moods, status effects,
inventories, relationships, traits, and user-defined categories. AI auto-update,
description prose cache, macros, slash commands, and prompt injection.

Install: copy or symlink this folder into `SillyTavern/public/scripts/extensions/third-party/state-referential/`.

See `docs/superpowers/specs/2026-05-23-tracker-plugin-v1-design.md` for the design.
